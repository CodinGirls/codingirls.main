# PyCon2014China 来也!

世界Pythoneer 的品牌盛会PyCon! 
由
[Python软件基金会](https://www.python.org/psf/about/)
下的[PyCon.org](http://PyCon.org)授权中国举办第四届
`PyConChina`大会。
本次大会由CPyUG、GDG、TopGeek在
上海、北京、珠海、杭州、苏州同时举办。
